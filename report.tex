\begin{abstract}
This report studies ground effect on a NACA 2412 airfoil using an updated \textbf{surface-panel method} (upper and lower surfaces are both modeled), with image panels for the ground wall. I present:
(1) $C_L$ vs. $\alpha$ for multiple $h/c$,
(2) $C_p$ vs. $x/c$ in the requested grouping (one subplot per $h/c$, all requested $\alpha$ in each panel),
(3) $\gamma$ distribution vs. $x/c$,
and (4) verification against XFOIL for no-ground cases.

I also keep the older camber-line figures in this report for side-by-side comparison.
\end{abstract}

\section{Objective}
The goal is to quantify how airfoil height from the ground changes:
\begin{itemize}
  \item lift coefficient $C_L$,
  \item pressure coefficient $C_p$,
  \item and circulation/sheet-strength trend ($\gamma$ proxy).
\end{itemize}

\section{Procedure and Methodology (Updated)}
\subsection{Airfoil and Cases}
\begin{itemize}
  \item Airfoil: NACA 2412.
  \item Chord: $c = 1$.
  \item Height cases: $h/c \to \infty$ (no wall), $1.00$, $0.50$, $0.15$.
  \item Angles:
  \begin{itemize}
    \item For $C_L$: $\alpha=-8^\circ$ to $12^\circ$,
    \item For $C_p$ and $\gamma$: $\alpha=-4^\circ,0^\circ,4^\circ,8^\circ,12^\circ$.
  \end{itemize}
\end{itemize}

\subsection{Surface-Panel Model}
The current solver is a Hess--Smith style 2D surface-panel method:
\begin{enumerate}
  \item Generate full NACA 2412 geometry (closed trailing edge): upper and lower coordinates.
  \item Split the full boundary into panels (cosine spacing in construction, then boundary panels).
  \item Unknowns are panel source strengths $\sigma_j$ and one global vortex strength $\gamma$.
  \item Enforce no-penetration at every panel control point.
  \item Enforce Kutta condition at trailing edge using upper/lower trailing-edge offsets.
  \item Solve one linear system for each angle of attack.
\end{enumerate}

\subsection{Panel-Method Formulation Used in the New Surface Code}
For each case ($h/c$ and $\alpha$), the solver uses:
\begin{itemize}
  \item $N$ constant-strength source panels ($\sigma_j$, $j=1,\dots,N$),
  \item one global vortex strength $\gamma$.
\end{itemize}

At each control point $i$, no-penetration is enforced:
\[
\sum_{j=1}^{N} A^{(\sigma)}_{ij}\sigma_j + A^{(\gamma)}_{i}\gamma
= -\mathbf{V}_\infty\cdot\mathbf{n}_i,
\]
where $A^{(\sigma)}_{ij}$ and $A^{(\gamma)}_{i}$ are normal-velocity influence
coefficients from source and vortex panels.

The trailing-edge Kutta condition is written as equality of tangential behavior
between upper and lower TE sides, giving one extra equation:
\[
\sum_{j=1}^{N} K^{(\sigma)}_j \sigma_j + K^{(\gamma)}\gamma
= -\mathbf{V}_\infty\cdot(\mathbf{t}_{TE,u}+\mathbf{t}_{TE,l}).
\]

So each solve is an $(N+1)\times(N+1)$ linear system:
\[
\begin{bmatrix}
\mathbf{A}^{(\sigma)} & \mathbf{A}^{(\gamma)}\\
\mathbf{K}^{(\sigma)T} & K^{(\gamma)}
\end{bmatrix}
\begin{bmatrix}
\boldsymbol{\sigma}\\ \gamma
\end{bmatrix}
=
\begin{bmatrix}
\mathbf{b}_n\\ b_K
\end{bmatrix}.
\]

After solving for $(\boldsymbol{\sigma},\gamma)$:
\begin{itemize}
  \item velocity at near-surface sample points is reconstructed,
  \item $C_p = 1-(V/U_\infty)^2$ is computed on upper/lower surfaces,
  \item total circulation is $\Gamma_{tot}=\gamma\sum_j \Delta s_j$ and
  $C_L=2\Gamma_{tot}/(U_\infty c)$.
\end{itemize}

\subsection{Ground-Effect Modeling}
For finite $h/c$, image panels are added across $y=0$:
\begin{itemize}
  \item source image has same sign,
  \item vortex image has opposite sign.
\end{itemize}
This enforces slip-wall behavior at the ground.

\subsection{Post-Processing}
\begin{itemize}
  \item Lift coefficient:
  \[
  C_L = \frac{2\Gamma_{\text{total}}}{U_\infty c},\qquad
  \Gamma_{\text{total}} = \gamma\sum_j \Delta s_j.
  \]
  \item Surface pressure coefficient from local velocity:
  \[
  C_p = 1 - \left(\frac{V}{U_\infty}\right)^2.
  \]
  \item The reported $\gamma$ plot is a useful circulation-trend diagnostic from upper/lower speed split.
\end{itemize}

\subsection{Verification Workflow}
No-ground model outputs are compared with XFOIL (Drela, 1989):
\begin{itemize}
  \item angles used: $\alpha = 0^\circ, 4^\circ, 8^\circ$,
  \item compare upper-surface $C_p(x/c)$,
  \item compute RMSE on overlap of the two $x/c$ grids,
  \item also compare $C_L$ values.
\end{itemize}

\subsection*{Why there is still a small difference now}
The difference is much smaller than before, but not exactly zero. This is expected, even with a surface-panel model:
\begin{enumerate}
  \item \textbf{Different numerical formulations:} this code uses one simple low-order panel setup (constant source strength on each surface panel plus one global circulation variable), while XFOIL uses a different panel and wake formulation with its own numerical details.
  \item \textbf{Trailing-edge and wake treatment are not identical:} small changes in Kutta enforcement and wake modeling can shift both local $C_p$ and integrated $C_L$.
  \item \textbf{Leading-edge sensitivity:} $C_p$ gradients are very steep near the leading edge, so small discretization differences create visible local error.
  \item \textbf{Grid/interpolation mismatch:} RMSE is evaluated after interpolating two different $x/c$ point sets and clipping near LE/TE, which adds small numerical differences.
\end{enumerate}

So the current model and XFOIL are now close, but they are still not mathematically identical solvers.

\section{Discussion and Inferences}
\begin{enumerate}
  \item $C_L$ increases with $\alpha$ for all heights.
  \item The strongest ground effect appears as we move the airfoil closer to the ground (as intuitively expected).
  \item The updated surface-panel model significantly reduces the XFOIL mismatch compared with the older camber-line model.
  \item A small residual error remains because the two solvers still differ in panel formulation, Kutta/wake treatment, and numerical details.
\end{enumerate}

\section{Model Limits}
\begin{itemize}
  \item This is a 2D inviscid potential-flow framework.
  \item No direct modeling of separation/stall physics.
  \item Ground is modeled as a slip wall using image panels.
\end{itemize}

\section{Conclusion}
After upgrading from camber-line to surface-panel modeling, the verification error against XFOIL is much smaller, which confirms the new equations are a better representation. At the same time, a small non-zero difference is still physically and numerically reasonable because the implementations are not identical.
